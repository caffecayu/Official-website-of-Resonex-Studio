## 一、项目上下文

《静默区》(SILENT AREA) 是一款以现代军事信息化战争为题材的回合制战棋游戏官网。网站承载游戏世界观展示、核心机制介绍、单位数据库、战役章节预览，以及社群关注入口。

详见 `PRODUCT.md`。

## 二、设计策略

### 场景设定

PLA 联合作战指挥中心，凌晨 3 点。战术显示屏发出幽蓝绿色辉光，军官们研读卫星情报和电子战读数。紧绷、专注、专业。

### 色彩策略 — Committed

基调从纯绿黑转向更深邃的蓝黑，主 accent 从荧光绿演化为磷光青绿（更接近真实战术显示屏的 P43 磷光），辅 accent 保留暖琥珀用于警告和高亮。

| Token | 旧值 | 新值 | 用途 |
|-------|------|------|------|
| `--bg` | `#080c0a` | `#060d13` | 页面底色 |
| `--panel` | `#111a15` | `#0b1620` | 面板底色 |
| `--panel2` | `#17251e` | `#101d2b` | 抬高面板 |
| `--text` | `#edf7ee` | `#e6eef3` | 正文 |
| `--muted` | `#a7b8aa` | `#7f97a5` | 辅助文字（已确保 WCAG AA ≥4.5:1） |
| `--accent` | `#9dff9f` | `#5eeadb` | 主强调色（磷光青绿） |
| `--accent2` | `#ffc76a` | `#f0a050` | 辅强调色（暖琥珀） |
| `--danger` | `#ff6b58` | `#e85545` | 威胁/警告 |
| `--line` | `rgba(174,255,193,0.18)` | `rgba(94,234,219,0.13)` | 分割线 |

### 排版

- 保持 Inter 系统字体栈，新增 `JetBrains Mono` 作为首选等宽字体
- 标题添加 `text-wrap: balance` 优化换行
- 收紧标题行高和字间距，增强军事印刷感
- 等宽字体仅在数据/代码/标签类元素使用

## 三、移除的 AI 痕迹

根据 impeccable skill 的 absolute bans：

1. **渐变文字** → `.signal-word` 从 `background-clip: text` 渐变改为纯色半透明水印
2. **每段 eyebrow** → 5 处减至仅 Hero 保留 1 处（重命名为 `.tac-kicker`）
3. **编号卡片** → 单位类型 01-05 改为战术缩写 INF / ARM / AVN / SUP / ART
4. **过度玻璃态** → header `backdrop-filter: blur(16px)` 减至 `8px`

## 七、设计原则（来自 PRODUCT.md）

1. **信息即武器** — 排版反映情报的层级与紧迫性
2. **暗中有光** — 暗色基调让关键信号更加突出
3. **真实胜于炫技** — 视觉语言追求克制与可信
4. **临场感优先** — 战术元素是空间锚点，不是装饰
5. **中文军事叙事的现代性** — 冷静、精准、技术自信
